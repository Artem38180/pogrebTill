Проект: классификация качества вина на основе характеристик
Технологии, использованные в проекте
Язык программирования
Python — основной язык реализации всего проекта.
Библиотеки для работы с данными
pandas — загрузка, анализ и предобработка табличных данных (чтение CSV, анализ структуры, обработка пропусков);
numpy — работа с числовыми массивами и математические операции.
Библиотеки машинного обучения
scikit‑learn (sklearn) — реализация алгоритмов классификации (логистическая регрессия, случайный лес, SVM), предобработка данных (StandardScaler), оценка качества моделей (метрики, матрица ошибок);
xgboost — реализация алгоритма градиентного бустинга (XGBoost).
Библиотеки визуализации
matplotlib — построение базовых графиков (в т. ч. диаграмм важности признаков);
seaborn — создание эстетичных визуализаций (тепловая карта матрицы ошибок, бар‑чарты).
Инструменты оценки моделей
Метрики классификации из sklearn.metrics:
accuracy_score (точность);
f1_score (F1‑мера);
confusion_matrix (матрица ошибок);
classification_report (детальный отчёт по классам).
train_test_split (из sklearn.model_selection) — разделение данных на обучающую и тестовую выборки.
Методы предобработки
StandardScaler (из sklearn.preprocessing) — нормализация числовых признаков;
LabelEncoder (из sklearn.preprocessing) — кодирование категориальных переменных (если присутствуют).
Алгоритмы машинного обучения
Логистическая регрессия (LogisticRegression);
Случайный лес (RandomForestClassifier);
Градиентный бустинг (XGBClassifier);
Метод опорных векторов (SVC).
Инфраструктурные инструменты
Jupyter Notebook / Google Colab (предполагается для интерактивной разработки);
Kaggle (источник датасета).
Форматы данных
CSV — формат исходного датасета.
Дополнительные возможности
optuna (опционально) — автоматизированный подбор гиперпараметров (в рамках дальнейшей оптимизации).
